If we can’t immediately live somewhere with mountains, lakes and all the things we really value, then I want us to find somewhere that feels interesting and a bit different, but is still affordable, that we can use as a base. I’m happy with that as a compromise, but it has to be very clearly temporary in my mind. I don’t want us drifting into treating it as permanent. It should be a stepping stone, somewhere that works for now while we figure out how to get to the place we actually want.
